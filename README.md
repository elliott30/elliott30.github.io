# elliott30.github.io

Personal profile site hosted on GitHub Pages.

Built using [Claude Code](https://claude.ai/code) — no frameworks, no build step, just plain HTML and CSS.

## Pages

- **Profile** (`index.html`) — About me, role, and areas of expertise
- **Repos** (`repos.html`) — Live feed of public GitHub repos via the GitHub API

## Stack

- Plain HTML + CSS
- GitHub API (client-side fetch for repos)
- GitHub Pages (auto-deployed from `main`)
